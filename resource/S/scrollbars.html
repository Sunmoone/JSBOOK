Summary

Returns the scrollbars object, whose visibility can checked.

Syntax

objRef = window.scrollbars

Example

The following complete HTML example shows way that the visible property of the scrollbars object is used.

<html>
<head>
  <title>Various DOM Tests</title>
  <script>
    var visibleScrollbars = window.scrollbars.visible;
  </script>
</head>
<body>
  <p>Various DOM Tests</p>
</body>
</html>
