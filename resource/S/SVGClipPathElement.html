SVGClipPathElement

The SVGClipPathElement interface provides access to the properties of <clippath> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable, SVGUnitTypes
Methods None
Properties
  • readonly SVGAnimatedEnumeration clipPathUnits
Normative document SVG 1.1 (2nd Edition)

Properties

Name Type Description
clipPathUnits SVGAnimatedEnumeration Corresponds to attribute clipPathUnits on the given <clippath> element. Takes one of the constants defined in SVGUnitTypes

Methods

The SVGClipPathElement interface do not provide any specific methods.
