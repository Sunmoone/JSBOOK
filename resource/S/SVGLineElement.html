SVGLineElement

The SVGLineElement interface provides access to the properties of <line> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable
