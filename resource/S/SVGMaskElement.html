SVGMaskElement

The SVGMaskElement interface provides access to the properties of <mask> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGUnitTypes

Methods

The SVGMaskElement interface do not provide any specific methods.
