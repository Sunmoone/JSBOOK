SVGStyleElement

The SVGStyleElement interface corresponds to the SVG <style> element.

Interface overview

Also implement SVGElement, SVGLangSpace
Methods None
Properties
  • readonly DOMString type
  • readonly DOMString media
  • readonly DOMString title
Normative document SVG 1.1 (2nd Edition)

Properties

Name Type Description
type DOMString Corresponds to attribute type on the given element. A DOMException is raised with code NO_MODIFICATION_ALLOWED_ERR on an attempt to change the value of a read only attribut.
media DOMString Corresponds to attribute media on the given element. A DOMException is raised with code NO_MODIFICATION_ALLOWED_ERR on an attempt to change the value of a read only attribut.
title DOMString Corresponds to attribute title on the given element. A DOMException is raised with code NO_MODIFICATION_ALLOWED_ERR on an attempt to change the value of a read only attribut.

Methods

The SVGStyleElement interface do not provide any specific methods.
