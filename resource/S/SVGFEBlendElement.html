feBlend element | SVGFEBlendElement object

Composites two objects together using commonly used imaging software blending modes. This method performs a pixel-wise combination of two input images.

Syntax

HTML
Copy 
<feBlend mode="normal" 
         in2="BackgroundImage" 
         in="SourceGraphic"/>
