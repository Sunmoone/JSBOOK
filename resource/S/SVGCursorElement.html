SVGCursorElement

The SVGCursorElement interface provides access to the properties of <cursor> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGURIReference, SVGTests, SVGExternalResourcesRequired
Methods None
Properties
  • readonly SVGAnimatedLength x
  • readonly SVGAnimatedLength y
Normative document SVG 1.1 (2nd Edition)

Properties

Name Type Description
x SVGAnimatedLength Corresponds to attribute x on the given <cursor> element.
y SVGAnimatedLength Corresponds to attribute y on the given <cursor> element.

Methods

The SVGCursorElement interface do not provide any specific methods.
