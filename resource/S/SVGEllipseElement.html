SVGEllipseElement

The SVGEllipseElement interface provides access to the properties of <ellipse> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable
