feTile element | SVGFETileElement object

Fills a target rectangle with tiled copies of an input image.

Syntax

Copy 
<filter id="tile1">
    <feImage x="0" y="0" width="10px" height="10px" result="raster1" xlink:href="../images/background.png"/>
    <feTile/>
</filter>

Examples

Creates multiple tiles of the saucer.svg graphic.

HTML
Copy 
<!DOCTYPE HTML>
<html>
  <head>
  </head>
  
  <body>
    <svg width="400" height="400">
      <defs>
        <filter id="MyFilter" filterUnits="userSpaceOnUse" x="0" y="0" width="400" height="400">
          <feImage x="0" y="0" width="100" height="100" xlink:href='saucer.svg' result='pict1'/>
          <feTile x="0" y="0" width="400" height="400" in="pict1"/>
        </filter>
      </defs>
      <use filter="url(#MyFilter)" x='0' y='0'/>				
    </svg>
  </body>
</html>
