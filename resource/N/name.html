window.name的应用  

  • 同一窗口打开页面
应用场景：网易娱乐中心（http://my.163.com/）

所有其他产品如果查看的内容为娱乐中心的某个TAB内容时均在同一窗口打开，同时在页面的TAB栏增加相关内容的标签
实现范例：
// 提醒用户该页面有变化，对于非标签浏览器可直接聚焦窗体
window.focus();
// 设置名称，其他产品用此名称可在同一窗口打开页面
window.name = '_neopen';
使用范例：
链接打开页面
<a href=”http://my.163.com/#tid=8999” target=”_neopen”>打开娱乐中心</a>
Javascript打开新窗口
window.open(‘http://my.163.com/#tid=8999’,’_neopen’);
  • 传递参数
应用场景：页面通过IFrame载入一个静态页面，而页面需要外面传入一些参数用来初始化，而又希望这个静态页面能使用浏览器的缓存而不受参数的影响，因此可以选择的方式之一就是使用name来传递参数。
使用范例：
<iframe src=”http://b.bst.126.net/poll.html” name=”u=aaa&c=bbbb”></iframe>
poll.html页面的javascript代码中使用window.name获取参数u=aaa&c=bbbb
  • 保存页面刷新前数据
应用场景：网易博客首页关于我模块未登录情况下发送消息
该应用的主要原理是页面刷新的时候window.name不会变化，应用此原理可以封装一个浏览器生命周期内的客户端缓存
  1. 当页面上一个完整流程中间存在一个步骤需要刷新页面的情况下需要保存用户刷新页面前的操作状态，其中方法之一就是使用window.name来记录页面刷新前的状态
  2. 用IFrame方式获取完全跨域的服务器数据的时候也可以使用此方式
