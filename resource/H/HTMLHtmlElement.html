HTMLHtmlElement

The HTMLHtmlElement interface serves as the root node for a given HTML document. This object inherits the properties and methods described in the HTMLElement interface.

You can retrieve the HTMLHtmlElement object for a given document by reading the value of the document.documentElement property.

Properties

Inherits properties from its parent, HTMLElement.

Name Type Description
version DOMString Version of the HTML Document Type Definition (DTD) that governs this document. This property should not be used any more as it is non-conforming. Simply omit it.

Methods

No specific method; inherits methods from its parent, HTMLElement.

Specifications

Specification Status Comment
WHATWG HTML Living Standard Living Standard No change from HTML5
HTML5 Candidate Recommendation The version element has been removed, as it is non-conforming.
Document Object Model (DOM) Level 2 HTML Specification Recommendation Reflecting the element change in HTML 4.01 Specification, the version property is now deprecated.
Document Object Model (DOM) Level 1 Specification Recommendation Initial definition
