接口 HTMLFontElement

  • 所有的接口：
    元素， HTMLElement，节点

    public interface HTMLFontElement
extends HTMLElement
    对字体的本地更改。请参见在 HTML 4.0 中的字体元素定义。此元素在 HTML 4.0 中已被否决。

    请参阅文档对象模型 (DOM) 级别 2 规格.
