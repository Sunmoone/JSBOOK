javascript中事件的拖放

目前javascript中有两种实现拖放的方法：系统拖放和模拟拖放。
系统拖放是利用操作系统来完成任务，支持的浏览器有IE \ Chrome \ firefox （《javascript高级程序设计》中说只有IE中支持，实际尝试了一下，目前除opera不支持，其他几种主流浏览器已基本实现该功能）。
1、拖动项事件
当某个项被拖动时依次触发dragstart----drag----dragend事件，可以使用ondragstart、ondrag、ondragend等事件来调用运行javascript代码。
dragstart事件触发后drag事件会一直触发，只要对象还在被拖动就不停止。当拖动停止时触发dragend事件。
下面这个来自《javascript高级程序设计》的例子将演示拖动时依次触发的事件，并将其显示在文本域内。

2、放置目标上触发的事件
当把对象拖放到有效的放置目标上时会触发dragenter事件，之后立即触发dragover，若拖动的对象最终放在目标对象的边界之内时触发drop事件，否则触发dragleave事件。

下面一个例子展示了将文本框1的内容拖放到文本框2中所依次触发的事件，将其显示在文本域内：
<html>
<head>
<title>System Drag and Drop exercise</title>
<script type="text/javascript">
var handleDrag = function(e)
{
var oText = document.getElementsByTagName("textarea")[0];
oText.value + =e.type+"\n";
}
</script>
</head>
<body>
<p>
<input type="text"  value="drag the text" ondragstart="handleDrag(event) ondrag="handleDrag(event)" ondragend="handleDrag(event)" />
</p>
<p>
<input type="text" ondragenter="handleDrag(event)" ondragover="handleDrag(event)" ondragleave="handleDrag(event)" ondrop="handleDrag(event)">
</p>
<p>
<textarea rows="15" cols="35" readonly="readonly"></textarea>
</p>
</body>
</html>
