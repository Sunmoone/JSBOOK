window.onclick

Called when the user clicks the mouse button while the cursor is in the window. This event is fired for any mouse button pressed; you can look at the event properties to find out which button was pressed and where.

Syntax

window.onclick =funcRef;

Parameters

  • funcRef is a reference to a function.

Example

window.onclick = doPopup;

<html>
<head>

<title>onclick test</title>

<script type="text/javascript">

window.onclick = clickPage;

function clickPage()
{
 alert("click event detected!");
}
</script>
</head>

<body>
<p>click and release the LH mouse button on this page.</p>
</body>
</html>

Notes

The click event is raised when the user clicks on the window.
