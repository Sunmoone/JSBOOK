Plugin

The Plugin interface provides information about a browser plugin.

Properties

Name Description Return Type Availability
Plugin.description A human readable description of the plugin. Read only. DOMString DOM 0
Plugin.filename The filename of the plugin file. Read only. DOMString DOM 0
Plugin.name The name of the plugin. Read only. DOMString DOM 0
Plugin.version The plugin's version number string. Read only. DOMString Gecko browsers only (Firefox 4+)

Methods

Plugin.item
Returns the MIME type of a supported content type, given the index number into a list of supported types.
Plugin.namedItem
Returns the MIME type of a supported item.
