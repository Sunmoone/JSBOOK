这是javascirpt内部使用的一个构造器,用来构造dom对象的属性，其实对象属性也是一个节点

var div = document.createElement('div');
div.setAttribute('id', 'foo');
div.attributes[0] instanceof Attr; // true

You cannot call it as constructor function, same as any other "constructor" functions of the DOM API (e.g. new HTMLDivElement()). You can only create an attribute via setAttribute.

Attr 对象

Attr 对象表示 Element 对象的属性。属性的容许值通常定义在 DTD 中。

由于 Attr 对象也是一种节点，因此它继承 Node 对象的属性和方法。不过属性无法拥有父节点，同时属性也不被认为是元素的子节点，对于许多 Node 对象的属性来说都将返回 null。
