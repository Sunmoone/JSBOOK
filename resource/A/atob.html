DOM 参考

概述

将已经被base64编码过的数据进行解码.关于Unicode和UTF-8编码字符串的解码

语法

    var decodedData = window.atob(encodedData);

例子

    var encodedData = window.btoa("Hello, world"); // 编码
    var decodedData = window.atob(encodedData); // 解码

备注

由于一些网络通讯协议的限制,你必须使用 window.btoa() 方法对原数据进行编码后,才能进行发送.接收方使用相当于 window.atob() 的方法对接受到的base64数据进行解码,得到原数据.例如, 发送某些含有ASCII码表中0到31之间的控制字符的数据.

在用javascript编写XPCOM组件时, atob() 方法也是可用的,虽然 全局对象已经不是window了.

规范

DOM Level 0 不属于任何标准.
