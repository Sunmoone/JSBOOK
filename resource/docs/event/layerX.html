Summary Returns the horizontal coordinate of the event relative to the current layer. Syntax event.layerX Examples var Xpos = event.layerX; Xpos is an integer value in pixels for the X coordinate of the mouse pointer, when the mouse event fired.

Notes layerX takes scrolling of the page into account, and returns a value relative to the whole of the document, unless the event occurs inside a positioned element, where the returned value is relative to the top left of the positioned element.
