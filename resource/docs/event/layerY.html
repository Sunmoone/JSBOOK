event.layerY Returns the vertical coordinate of the event relative to the current layer. Syntax var Ypos = event.layerY; Ypos is an integer value in pixels for the Y coordinate of the mouse pointer, when the mouse event fired.

Notes layerY takes scrolling of the page into account, and returns a value relative to the whole of the document, unless the event occurs inside a positioned element, where the returned value is relative to the top left of the positioned element.
