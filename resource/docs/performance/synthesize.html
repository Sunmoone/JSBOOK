对资源文件加载情况的统计 这是整个页面加载的时间统计,如果要统计其他信息,比如页面上所有的资源文件,比如css和图片文件的分段加载的统计怎么办呢? HTML5 草案中还有另一个 Resource Timing 统计,草案看这里:http://w3c-test.org/webperf/specs/ResourceTiming/ 可惜的是,目前没有任何浏览器实现这个接口,所以目前只能先了解一下,将来出来之后再应用. 传统浏览器的统计 对于没有实现这个接口的浏览器,就只能采用传统方法,在头部计算时间戳,在 window.onload 和 domReady 的时候再计算一个时间戳,计算一下差值.具体实现方法就不用讲了吧.
