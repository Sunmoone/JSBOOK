google closure 是google推出的前端框架和工具组
有4个部分，
1. compiler，用来编译js代码。
2. library，有点像.net或者java的标准库，
里面有页面控件对象和其他组建，
3. template，这个部分是用来做js和html分离的，
概念有点像drupal里的theme的感觉，其中用compiler可以编译成js或者java对象
4. linter，用来follow代码风格的，作用不大，除非多人合作开发，想统一一下风格的时候
前端框架要解决4个问题
1. 兼容性
2. HTML与script分离
3. 浏览器行为支持
4. 闭包（常见于ajax的回调问题）
