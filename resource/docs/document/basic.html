document介绍

不要把Document和document搞混他们是不一样的，不就是Document构造出来的吗？不是这样的

document不是Document的一个实例对象

其实document就是整个文档的对象，html标签代表根标签页面也就是整个文档但是跟document对象是不一样的，document跟其他元素的的继承对象不一样，单独为document编写了接口

document是有HTMLDocument构造出来的

HTMLDocument继承Documen,Document继承Node,Node继承Object

而其他的元素都是有相应的HTML(XX)Element元素构造出来,然后继承HTMLElement,HTMLElement继承Element,Element继承Node,Node继承Object

document 表示的是一个文档对象，windows 表示一个窗口对象。

html是HTMLHtmlElement的对象实例继承HTMLElement继承Element继承Node继承Object
