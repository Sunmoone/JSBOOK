概述

返回当前窗口中包含的框架数量(框架包括frame和iframe两种元素).

语法

framesCount = window.length;
  • framesCount就是该窗口中框架的数量.

示例

if (window.length) {
  // 该窗口包含至少一个子框架
}

规范

DOM Level 0 不属于任何标准.
