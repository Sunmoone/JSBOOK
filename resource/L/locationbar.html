定义：locationbar属性允许你引用的地址栏（或地址栏）中的浏览器，以测试它是否是可见或不可见。 与大多数浏览器，这是一个只读字段。 这不是一个保留字，因此您可以声明自己的变量或函数调用locationbar但如果这样做，那么你将不能够访问浏览器的一部分。

窗口对象的locationbar、菜单栏和滚动条属性返回"BarInfo"对象的引用。

语法window.locationbar
window.menubar
window.scrollbars

此代码示例显示如何评价栏在浏览器窗口中的存在。

Javascript 的代码示例
< 脚本类型 ="文本/javascript">
if(window.locationbar) {
if(window.locationbar.visible) {
alert ("此窗口中有一个地址栏")。
} 其他 {
alert ("此窗口没有地址栏.")。
    }
} 其他 {
alert ("此浏览器不支持的 locationbar 属性.")。
}
</脚本 >

提示:通过上面插入"菜单栏"或"滚动条"代替"locationbar"，可以使用相同的代码。
